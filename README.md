# Linkedin Clone

A Linkedin Clone built using React.js, Redux, Material-UI and firebase.
`react-flip-move` is used for animation.

## Installation

Install with npm

```bash
  git clone https://github.com/harikanani/Linkedin-Clone.git
  cd Linkedin-Clone
  npm install
```

## Tech Stack

**Client:** React, Redux

**Hosting:** Firebase

## Deployment

To deploy this project run

```bash
    yarn build
    firebase login
    firebase init
    firebase deploy

```
